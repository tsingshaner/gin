## 响应体结构

响应体使用 `JSON` 格式，结构如下：

请求成功响应体 `body`

```json
{
  "code": "XXXXXX",
  "data": {}
}
```

请求错误响应体 `body`

```json
{
  "code": "XXXXXX",
  "err": "XXXXXX"
}
```
